### 项目运行步骤
#### 环境要求
确保已经具备JDK（推荐版本为JDK17）、Maven （v3.6.3 ~ v3.8.4）、Mysql （v5.6 +）、Node.js（v16.13.1~v18.20.3）。
#### 后端
使用IDEA打开be目录，加载完`pom.xml`的依赖后，直接运行`Application`类即可。
#### 前端
前端在fe目录下开一个终端，执行`npm i`安装完成依赖，然后`npm run dev`即可。
